# Sikkhayon Academy Backend

### This is Sikkhayon Academy Backend server where we provide the API's and other server side features for our system

### Technologies Used:

- Node.js
- Express.js
- Npm
- Vercel

### How to run:

```sh
  npm install
  npm start
```

We have used vercel to deploy our server application.
